Consentful Interface

Link: https://editor.p5js.org/koj097/sketches/N7yZWtaY3

Description

A simple yes or no button with some personality to the questions and answers it gives. If lays out all the consequences of proceeding with the interface, and lets the user make a choice.

Design Process

Writing the choice branch

Illustrating the Choice Branch the choice branch

Reflection

I wanted to do something simple and related to buttons. So I decided to make a short, interactive sort of warning for the viewers to have a bit of fun with. The interface responds to F.R.I.E.S. by making sure the user was enthusiastic, free, and knows the consequences of interacting with said interface.
